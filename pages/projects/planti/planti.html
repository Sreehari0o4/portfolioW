Overview

Civic Eye is a mobile application that allows citizens to report traffic rule violations they witness in their surroundings—such as illegal parking, helmetless riding, signal jumping, or wrong-way driving. The app empowers users to upload photographic evidence and submit brief details of the violation.
A key innovation in this project is the integration of a Large Language Model (LLM) that generates automated challans based on the nature of the violation. Once verified, a percentage of the fine collected is rewarded to the reporting citizen, encouraging active civic participation and traffic discipline.

Problem Statement

Reporting civic issues is often tedious and unrewarded. Citizens are either unaware of where to report or demotivated due to lack of acknowledgment. Civic Eye addresses this by:

  • Offering a simple reporting interface
  • Using an LLM (Large Language Model) to auto-generate challans for valid cases
  • Rewarding the reporting citizen with a percentage of the fine

Tools & Technologies

Challenges and Solutions

  • Challenge: Understanding Violation Context for Challan Generation
    Solution: Integrated a custom prompt with an LLM that extracts the type of violation and generates a traffic challan in proper legal format.
  • Challenge: Verifying Report Authenticity
    Solution: Created a tagging system to allow backend or admin review before the challan is issued and reward is unlocked.
  • Challenge: Secure Media and Data Handling
    Solution: Used Appwrite’s file storage and database with authentication to ensure data protection and proper user tracking.
  • Challenge: Incentivizing without Over-complication
    Solution: Used a simple reward flag system tied to successful challan generation, viewable in the user’s report history.

Outcome and Result

  • Delivered a fully functional MVP for traffic violation reporting
  • Successfully generated challans dynamically with LLMs during test runs
  • Positive feedback for the reward system — users found it motivating and fair
  • System prepared for potential scaling with admin dashboards, law enforcement APIs, or payment integration
